BIP 39 secret splitter

This tool uses the one-time pad cryptographic method to split a BIP 39 seed phrase into multiple encrypted key pairs (each consisting of an A and B key). A complete pair (e.g. A1 and B1) must be combined to decrypt the secret. Each key in a pair contains no information about the other key.

This is designed to work offline and has 0 dependencies on any external libraries.

Warning: you are online! For security, it is recommended that you disconnect from the internet before inputting your seed phrase!

Storing

The encrypted keys (A1, A2, B1, and B2) should go into 3 locations, with the following arrangement of keys:

  • Location 1: A1, A2
  • Location 2: A1, B2
  • Location 3: B1, A2

This way, you can ensure that any two locations can produce a complete key pair and unlock the entire secret.
